# COMP4037_CW2_DangWeiwei
README for Heatmap Visualization Project

Project Overview
This repository contains a Jupyter Notebook project designed to visualize hospital episode statistics through heatmap generation. The project processes admission and diagnosis data spanning from 2015 to 2024.

Repository Contents
The project consists of three core files:
- heatmap.ipynb: The primary Jupyter Notebook script containing the logic for data processing and heatmap visualization.
- hosp-epis-stat-admi-diag-2015-2024.xlsx: The source dataset containing hospital episode statistics, admission details, and diagnosis records. This file serves as the input for the notebook.
- heatmap.html: A standalone HTML file representing the rendered output of the notebook, allowing for interactive viewing of the generated heatmaps without requiring a Python environment.

Usage Instructions
To reproduce the visualization or modify the analysis, please follow these steps:
1. Prerequisites: Ensure you have Jupyter Notebook or JupyterLab installed on your machine. You can install it via pip if necessary.
2. Execution:
   - Clone or download this repository to your local environment.
   - Launch the Jupyter environment in the directory containing these files.
   - Open heatmap.ipynb.
   - Run all cells to process the data from the Excel file and generate the visualizations.

Viewing Output
- Once executed, you can view the results directly within the notebook.
- Alternatively, open heatmap.html in any modern web browser to view the static or interactive output.

Data Source
The analysis relies on the provided Excel file (hosp-epis-stat-admi-diag-2015-2024.xlsx). Ensure this file remains in the same directory as the notebook to avoid file path errors during execution.
